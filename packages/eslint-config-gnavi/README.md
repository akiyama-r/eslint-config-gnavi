> Gnavi's [ESLint](http://eslint.org/) shareable config for JavaScript

## Install

```sh
$ npm i --save-dev eslint-config-gnavi
```

## Usage

In your `.eslintrc`:

```
{
  "extends": ["eslint-config-gnavi"]
}
```

or in your `package.json`:

```json
"eslintConfig": {
  "extends": ["eslint-config-gnavi"]
}
```
